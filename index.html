p5.js Cheat Sheet

Color

background( gray )
background( r, g, b )		 changes the background color
fill( gray )
fill( r, g, b )		 changes the color of a shape
noFill() removes the inside of a shape
stroke( gray )
stroke( r, g, b )		 changes the border color of a shape
noStroke() removes the border of a shape
strokeWeight( size ) changes the thickness of a shape’s border

Shapes

square( x, y, size ) draws a square
rect( x, y, width, height ) draws a rectangle
circle( x, y, diameter ) draws a circle
ellipse( x, y, width, height ) draws an ellipse
line( x1, y1, x2, y2 ) draws a line
triangle( x1, y1, x2, y2, x3, y3 ) draws a triangle

Images and Video

loadImage( “file-name.jpg”, x, y, width, height ) loads an image file, goes in the setup() function
createCapture( VIDEO ) activates your webcam, goes in the setup() function
image( name, x, y, width, height ) displays an image or video

Structure

setup() runs once, used to program the settings
draw() after setup(), it loops forever until the program is stopped or you call noLoop()
preload() optional, runs once before setup(), used to load files

How to read this reference

Parameter What does it mean? What can the value be?
gray a shade of gray number from 0 to 255
r g b red, green, blue numbers from 0 to 255
x horizontal position any number
y vertical position any number
size size of something in pixels any number
width width of a shape any number
height height of a shape any number
diameter size of a circle from edge to edge any number
x1 y1
x2 y2...		 format used when you need multiple xy points any numbers
name variable name for an image or video name of a variable
VIDEO keyword to use the webcam VIDEO
